# Just a static site for my so-called portfolio website

Using Vite with plugins for 
- Tailwind
- Handlebars 

## Structure

- `index.html` — Main landing page
- `src/` — Styles, scripts, and Handlebars partials
- `public/` — Static assets and T-Rex game (`trex.html`)
- `package.json` — Project metadata and scripts
- `vite.config.js` — Vite build configuration

## Development

To run locally:

```sh
npm install
npm run dev
```

To build for production:

```sh
npm run build
```
